# What MQTT? 

MQTTって一体なんなのか？  
ブラウザで実行しているアプリをMQTTクライアントとするために、`MQTT on websocket`したサンプルです。

# demo

<http://mitsuruog.github.io/what-mqtt/>

# install

```
git clone https://github.com/mitsuruog/what-mqtt.git
cd what-mqtt
npm install
npm run start
```

Check it. <http://localhost:3000>
